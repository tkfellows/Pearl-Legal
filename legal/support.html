Frequently Asked Questions

How do I backup my data in Pearl?
Pearl offers multiple backup options: 1) Enable iCloud backup in the app settings for automatic cloud backup, 2) Use the export feature to create local backups in JSON, CSV, or XML format, 3) Use your device's native backup features. We recommend using multiple backup methods for maximum data safety.
Is my medical data secure?
Yes, Pearl is designed with privacy-first principles. All data is stored locally on your device by default. iCloud backup is optional and uses Apple's end-to-end encryption. No medical data is transmitted to external servers without your permission. We recommend using device security features like Face ID, Touch ID, or passcodes.
How accurate is the speech-to-text feature?
Pearl uses Apple's Speech Framework for transcription, which is highly accurate for medical terminology. However, we recommend always verifying transcriptions for medical accuracy before relying on them in clinical settings. The app includes disclaimers about transcription accuracy and the need for verification.
Can I use Pearl in my medical institution?
Pearl is designed for individual healthcare professionals and complies with medical privacy standards. However, institutional use should be approved by your IT and compliance teams. We provide compliance documentation to assist with institutional review processes. Contact us for institutional deployment guidance.
How do I organize my medical pearls effectively?
Use the tagging system to categorize pearls by medical specialty, procedures, or clinical presentations. Create consistent tag naming conventions. Use the search feature to find pearls quickly. Consider using the dual-save photo system to preserve original images while creating cropped versions for display.
What if I lose my device?
If you have iCloud backup enabled, your data will be restored when you reinstall Pearl on a new device. If you don't have iCloud backup, you can restore from exported backup files. We recommend regular exports and iCloud backup for maximum data protection. Contact support immediately if you need assistance with data recovery.
